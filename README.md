## Description
A console-based Movie Ticket Booking System developed using Java and MySQL.   
The application supports Admin and User roles. Admins can manage movies
and view bookings for their theatre, while users can search movies,
book tickets, and view booking history.

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Modules
### Admin Operations
- Signup and Login
- Add movies for a theatre
- Remove movies
- View bookings for own theatre

### User Operations
- Signup and Login
- Search movies by name
- Search movies by theatre
- View all available movies
- Book movie tickets
- View booking history

## How to Run
1. Clone the repository
2. Import project into Eclipse
3. Add MySQL Connector JAR to Build Path
4. Run the SQL file to create database and tables
5. Update DB credentials in `Sql.java`
6. Run `Main_Program.java`

## Project Type
Console-based Java application using JDBC.
